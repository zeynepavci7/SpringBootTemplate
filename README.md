A basic RESTful task API using Java and Spring Boot.

## Features
- Create a task
- List all tasks
- Get task by id
- Update a task
- Delete a task

## Technologies
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## API Endpoints
- POST /api/tasks
- GET /api/tasks
- GET /api/tasks/{id}
- PUT /api/tasks/{id}
- PATCH /api/tasks/{id}/complete
- DELETE /api/tasks/{id}

## Testing
All CRUD (Create, Read, Update, Delete) endpoints were tested using Postman and verified to be working successfully.
